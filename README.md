# ✍️ Instagram Post Generator - @losescritosdedani

Este es un proyecto personal desarrollado para automatizar y estandarizar la creación de contenido visual para la cuenta de Instagram **@losescritosdedani**.

## 🚀 Tecnologías utilizadas
* **HTML5/CSS3**: Estructura y diseño basado en tipografía serif y minimalismo.
* **JavaScript (ES6)**: Lógica de manipulación del DOM.
* **html2canvas**: Librería para la exportación de nodos DOM a imágenes de alta resolución (PNG).

## 🛠️ Características
* **Fidelidad Visual**: Réplica exacta del branding (colores `#9ba7b5` y tipografía *Playfair Display*).
* **Escalabilidad**: Sistema de plantillas que permite crear nuevos posts en segundos.
* **Exportación Directa**: Botón integrado para descargar el post en formato 1080x1350px (4:5 ratio).

## 📦 Instalación y Uso
1. Clona el repositorio: `git clone https://github.com/tu-usuario/nombre-repo.git`
2. Abre `index.html` en tu navegador (se recomienda usar Live Server en VS Code).
3. Modifica el texto en los archivos dentro de `/posts`.
4. Haz clic en "Descargar PNG".
